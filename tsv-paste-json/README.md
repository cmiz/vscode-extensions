# TSV Paste JSON

クリップボードのTSVテキストをJSONに変換してエディタにペーストします。

![screenshot](https://user-images.githubusercontent.com/69108539/110209693-19bccb80-7ed1-11eb-87a3-31c31f9c0d71.gif)

# Development

## Create VSIX

```
npm i
vsce package
```

## Install from VSIX

- VSCodeのコマンドパレット(F1)
- Extensions: Install from VSIX... を選択
- .vsix ファイルを選択
